# AsistenQ

Marketplace digital untuk tools, ebook, video, dan kelas.

## Fokus Awal
- Tools editing video
- Tools untuk kebutuhan YouTube
- Kelas dan konten edukasi premium

## Mode Langganan
- Tools: bulanan
- Kelas premium: tahunan

## Development Lokal
1. Salin file konfigurasi dari `.env.example`
2. Buka `index.html` di browser untuk prototype awal
3. Kembangkan backend dan admin panel dari struktur ini
